# EMAS - Encapsulation of methods with arbitrary signatures
### Запуск:
Открыть проект
Нажать F5 
if успешно: с кайфом
else:
Зайти в свойства проекта - Свойства конфигурации - Стандарт языка С++ - Выбрать "Стандарт ISO C++17" - Нажать F5
### Что должно работать:
1. Метод через обёртку
2. Методы с 0, 1, 2, 3 параметрами
3. Обработка ошибок
4. Многопоточность
5. Значения по умолчанию
