# Алгоритм Хаффмана
## Запуск
javac *.java
java HuffmanTest
## Использование
java Main режим входной_файл выходной_файл
## Примеры
#### java Main encode document.txt compressed.bin - кодирование
#### java Main decode compressed.bin document_restored.txt - декодирование
## Описание структуры закодированного файла (на примере 11111111112222233333)
```
Содержимое файла: 11111111112222233333
Частоты символов:
1 - 10 раз
2 - 5 раз
3 - 5 раз
Коды Хаффмана:
'1' - "0"    (1 бит)
'2' - "10"   (2 бита)
'3' - "11"   (2 бита)
Битовая строка после кодирования:
1111111111 22222 33333
0000000000 10101 01011 11111111
Итого: 30 бит = 0000000000 1010101011 1111111111
```
## Восстановление 
```
Содержимое: 000000000010101010111111111111
3 символа:
'1'(10 раз)
'2'(5 раз)
'3'(5 раз)
Коды Хаффмана:
'1'="0"
'2'="10"
'3'="11"
Длина: 30 бит
Преобразуем в биты: 00000000 00101000 11111111 11000000
Берем первые 30 бит: 00000000 00101000 11111111 11
Декодируем:
0 → '1' (10 раз)
10 → '2' (5 раз)
11 → '3' (5 раз)
Результат: 11111111112222233333 ✓
```
